# Fiche Poulets de Poulailler · 3,5 kg

Mini-fiche commerciale HTML prête à publier sur GitHub Pages pour vendre des poulets de poulailler.

## Offre

- Produit : poulets de poulailler
- Poids moyen : 3,5 kg
- Prix : 3 500 FCFA l’unité
- Contact direct / WhatsApp : +221 77 832 96 12

## Objectif

Cette page sert de fiche simple, rapide et partageable pour permettre aux clients de contacter directement le vendeur par téléphone ou WhatsApp.

Elle peut être partagée par lien, QR code, WhatsApp, SMS ou réseaux sociaux.

## Fichier principal

Le fichier principal du site est :

```text
index.html
```

Si le fichier téléchargé s’appelle `fiche-poulets-3kg5-index.html`, il faut le renommer en `index.html` avant de le déposer dans le dépôt GitHub.

## Publication sur GitHub Pages

1. Créer un dépôt GitHub, par exemple :

```text
fiche-poulets-3kg5
```

2. Déposer le fichier `index.html` à la racine du dépôt.

3. Aller dans :

```text
Settings → Pages
```

4. Dans **Build and deployment**, choisir :

```text
Deploy from a branch
```

5. Sélectionner :

```text
Branch: main
Folder: /root
```

6. Enregistrer.

GitHub donnera ensuite une adresse publique du type :

```text
https://nom-utilisateur.github.io/fiche-poulets-3kg5/
```

## Boutons inclus dans la fiche

La fiche contient :

- un bouton WhatsApp direct ;
- un bouton appel direct ;
- un message WhatsApp prérempli ;
- les informations de prix et de poids ;
- une présentation adaptée au téléphone.

## Message WhatsApp prérempli

```text
Bonjour, je viens pour les poulets de 3,5 kg à 3 500 FCFA. Est-ce encore disponible ?
```

## Conseil terrain

Pour une meilleure circulation, générer un QR code vers l’adresse GitHub Pages finale, puis le partager sur WhatsApp, statut, affiche ou impression papier.
